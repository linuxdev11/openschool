### [en] description
openschool is a lightweight and open-source electronic diary system built with fastapi. this project was created as a free alternative to closed government systems. under the hood: asynchronous operations and bootstrap for styling. currently, it's not very functional, but i will develop it (no promises).

#### current features:
* **auth**: user authorization for students and teachers.
* **data management**: viewing and adding grades via a clean ui.
* **architecture**: multiple sqlite database support for data isolation.
* **easter egg**: "i'm a teapot" status code (418) with secret developer content.

#### roadmap:
* **v1.0**: stable prototype in python (fastapi).
* **v1.5**: hybrid architecture (python + c#).
* **v2.0**: complete migration to asp.net core for maximum reliability and typing.
